## 🌐 Deploy to Vercel (Easiest)

### Step 1: Push to GitHub
```bash
cd portfolio
git init
git add .
git commit -m "My awesome portfolio"
git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
git push -u origin main
```

### Step 2: Deploy
1. Go to [vercel.com](https://vercel.com)
2. Sign in with GitHub
3. Click "New Project"
4. Select your portfolio repo
5. Click "Deploy"

**That's it!** Your portfolio is live in 30 seconds ✨

Your URL will be something like: `https://portfolio-xyz.vercel.app`

### Step 3: Connect Custom Domain (Optional)
In Vercel dashboard → Settings → Domains
- Add your domain (e.g., `praveen.dev`)
- Follow DNS instructions
